Welcome to 7Bot ScratchX Extension Page.

This extension lets you control the 7Bot Arm using the Scratch visual programming platform. The extension communicates with the 7Bot Arm using the SystemSoftware communication protocol.

The 7Bot extension runs on the ScratchX experimental platform for Scratch and requires the 7Bot robotic arm from 7Bot.cc.
